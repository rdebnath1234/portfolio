# Portfolio Project

A full-stack portfolio website built with **React**, **Node.js**, and **MongoDB**.  
Showcases projects, contact form, and modern web development skills.

---

## Features

- Responsive design
- Portfolio showcase
- Contact form with backend API
- Dark mode toggle
- MongoDB database integration
